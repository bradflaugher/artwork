# Flowchart of a Quiet Life

```
                  [START: a quiet life]
                          |
                  Was it quiet?
                    /        \
                  yes         no
                  /              \
        [keep it quiet]      [make it quieter]
                  \              /
                   \            /
                    \          /
                  [is it quiet now?]
                       /      \
                     yes       no
                      |         |
                      |    [remove one (1) sound]
                      |         |
                      |    [remove one (1) self]
                      |         |
                      |    (loop: up to 60 years)
                      |         |
                      \        /
                       \      /
                   [is it quiet?]
                      /       \
                    yes        no
                    /            \
          [stay]            [leave quietly]
            |                      |
            |               [did anyone notice?]
            |                 /          \
            |               yes           no
            |              /                \
            |     [you were not quiet]  [you were quiet]
            |              \                /
            |               \              /
            \                \            /
             \                \          /
              \                \        /
               \               [END: a quiet life]
                \                      |
                 \               [no one remembers
                  \               whether you started]
                   \                     |
                    --------> [STOP] <----
```

**Legend:**
- ▢ a decision you will not know you made
- ▣ a sound you were not supposed to make
- ◯ the place where the chart forgets you

**Note:** The flowchart has no terminal node that resolves the question of whether the quiet life was worth it. This is intentional. A flowchart that could answer that would be a different document, called *a eulogy*, and would be drawn by someone else, from memory, which is never quiet, and never accurate, and is the only part of you that survives the chart.

> Epigraph: *A quiet life is a flowchart with the labels rubbed out. You walk it anyway.*
