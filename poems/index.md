# Strange Poems — an anthology

A growing cabinet of linguistic oddities. Not a poetry journal. Not a literary movement. A place where text behaves like artwork: wrong grammar on purpose, impossible objects, bureaucratic surrealism, procedural nonsense, constraint forms that feel broken.

## How to read this

Every poem is a single `.md` file in this directory. Each file has a title, a body, and a one-line gloss or epigraph. Read them in any order; the batches describe a flavor of strangeness, not a sequence.

## Naming convention

Filenames are `kebab-case-title.md` — lowercase, hyphen-separated, no version numbers, no dates. A filename should sound like a title, not a filename.

## The three strangenesses

### Batch 1 — surreal / liminal
Impossible geography, dream-logic syntax, objects that shouldn't speak. The strangeness lives in the image: a hallway that opens onto a hallway, a verb that refuses its subject.

### Batch 2 — constraint / formal
Broken sonnets, bureaucratic forms as verse, numbered inventories, legalistic odes. The strangeness lives in the shape: a form that is also a poem, a poem that is also a form.

### Batch 3 — absurd / procedural
Algorithmic lists, mistranslation cascades, instructions that aren't instructions. The strangeness lives in the procedure: a recipe that cooks nothing, a loop that terminates in a feeling.

## Adding a poem

Create one `.md` file in this directory. Give it a curator-style filename. Put a title, the body, and a one-line epigraph inside. Then add a link under the correct batch above. An unlinked poem is a lost poem. See `../AGENTS.md` for the full rules.
