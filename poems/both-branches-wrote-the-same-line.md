# Both Branches Wrote the Same Line

```
<<<<<<< HEAD
We were always going to stay in this room.
The radiator hums in a key of low G.
=======
We were always going to leave by the window.
The radiator has been cold since November.
>>>>>>> incoming-history
```

The compiler is not angry; it is merely patient.
It presents both versions of the afternoon
and waits for the editor to perform the surgery.

If you choose the window,
the room becomes a file we deleted.
If you choose the room,
the window is just a rectangle of glass
with nothing behind it.

You can also delete the markers
and keep both sentences,
but then the room is twice as crowded,
and we must sleep on top of our own ghosts,
who have different memories of the rent.

```
git merge --abort
```

is a way of saying:
*Let us pretend the conversation never happened.
Let us go back to the moment before the collision,
when we were still in our own branches,
running in parallel,
believing in the same main branch
that has already moved on without us.*

> Epigraph: *A conflict marker is a boundary that remembers both sides of the argument before one is erased.*
