# The Wind Returns Things to the Wrong Sky

The wind does not arrive. The wind is the leaving
of somewhere you were never told about,
a province with no flag, only a direction
that keeps changing its mind in your hair.

It borrows the voice of the field
and gives it back to the wrong field.
It lifts a name off a mailbox
and reattaches it, gently, to a different house,
so that for one afternoon
two families answer to the same wind.

We sent a kite up to ask it questions.
The kite came down speaking a slight accent,
pulling north when we pulled,
insisting the string belonged to it now.

The wind has been carrying something for years —
we can hear it knock against the inside of the gust —
but it will not set the parcel down,
and it has forgotten, the way a courier forgets,
which sky the thing was addressed to,
which window, on which street, was meant to open
and take delivery of all this moving air.

So it keeps everything aloft.
So it gives the dust back to the dust
at a slightly wrong altitude,
the way you return a borrowed grief
to the wrong friend, and they accept it,
because the wind is convincing,
because the wind has always sounded like it knows.

> Gloss: *The wind is a courier that has lost the address but not the parcel.*
