# The Fog Files Its Report

The fog arrives before the town agrees to it,
already holding a clipboard,
already counting what it intends to subtract.

It takes the church steeple first,
not because the steeple matters
but because removals look more serious
when they begin with something tall.

The fog does not erase the streetlights.
It interviews them. One by one
it asks each lamp what it has seen,
writes the answer in a hand no one can read,
and files the lamp under *pending*.

By morning it returns the town
the way a clerk returns a form:
stamped, slightly damp,
with one field left deliberately blank.

We never learn which field.
We only notice, for a week afterward,
that the church has no steeple in our memory,
that we describe the morning
as a thing that happened to someone else
and was reported to us, later, in summary.

> Epigraph: *Visibility is a privilege the fog reviews quarterly.*
