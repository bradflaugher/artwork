# AGENTS.md

Rules for agents and contributors working in this repo.

## What this repo is

A home for **artwork** — visual pieces, manifestos, series, and conceptual work. Not a software project. Treat every change as a curatorial act.

## Core rules

1. **Original work only.** Submit art you created. No reproductions, no unattributed appropriations, no scraped or model-mimicked copies of living artists' signature styles. If you reference an existing work, transform it enough to stand on its own and name the reference.

2. **Subversive intent is the point.** This collection favors work that challenges, unsettles, refuses, or subverts — addictive interfaces, surveillance aesthetics, algorithmic attention, sanitized politeness. Pretty is fine; pretty *and* dangerous is better. Decorative filler belongs elsewhere.

3. **Art first, code second.** Markdown, images, and text are first-class. Add code only when it *is* the medium. Don't scaffold frameworks, build systems, or tooling unless the art demands it.

4. **Keep the gallery clean.** Each series lives in its own directory with a manifesto or index. Name files like a curator, not like a compiler (`the-infinite-scroll.png`, not `img1_final_v3.PNG`). No build artifacts, caches, or scratch files committed.

5. **Honor the license.** Everything here is MIT (see `LICENSE`). Contribute with that in mind — your work will be freely shared, modified, and redistributed. Attribution stays; permission is granted.

6. **Say what the work means.** Every piece ships with an artist statement or caption. A subversive image without context is just an image. The words are part of the piece.

7. **Small, complete contributions.** One coherent series or piece per change. Finish it — image, statement, index entry — before opening a contribution. No half-installed exhibitions.
